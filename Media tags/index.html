HTML Media Tags

  • HTML video Element

    The HTML <video> element is used to show a video on a web page.

    Example: To show a video in HTML, use the ≪video> element

    <video src="video.mp4" width="320px" height="240px" controls autoplay muted loop><video>


    • The source element allows you to specify alternative video files which the browser may choose from. the browser will use the first recognized format.
    • Itis good idea to always include width and height attributes. if height and width are not set, the page might flicker while the video loads.
    • The controls attribute adds video controls, like play, pause, and volume.
    • autoplay attribute helps to start a video automatically.
    • Add muted after autoplay to let video start playing automattically

  • HTML audio Element

    To play an audio file in HTML, use the <audio> element

    Example:

    <audio src="s.t.mp3" controls> </audio>
