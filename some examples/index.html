HTML basics:

  1. What is HTML?

    • The Hyper Text Markup Language or HTML is the standard markup Language for document designed to be displayed in a web browser. HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets. Tags such as < img > and <input > directly introduce content into the page. Other tags such as < p > surround and provide information about document text and may include other tags as sub-elements. Browsers do not display the HTML tags but use them to interpret the content of the page.

    • Before creating any web page we should add an boilerplate template.

    • Currently, HTML5 is the latest and most enhanced version of HTML.

  2. What is Boilerplate?

    • Boilerplate is a starter code which contains important information like the docktype, metadata external stylesheet and script tags.

    • As we can see above there are some tags like docktype, html, head, meta, title etc.

  3. What is a docktype in HTML?

    • docktype tells the browser what version of HTML the page is written in. if we forget to include this line of code, then some of the HTML 5 tags like, < article >, < header > and <footer> may not be supported by the browser.

  4. What is the HTML root element?

    • The < html > tag is the top level element of the HTML file. You will nest the < head > and < body > tags inside of it. The lang attribute inside the opening tag sets the language for the page. It is also good to include it for accessibility reasons, because screen readers will know how to properly pronounce the text.

  5. What are head tags in HTML?

    • The < head > tags contain information that is processed by machines. Inside the < head > tags, you will nest metadata which is data that describes the document to the machine.

  6. What is UTF-8 character encoding?

    • UTF-8 is the most common character encoding method used on the internet today, and is the default character set for HTML5. This will usually be the first <meta > tag shown in the < head > element.

  7. What is the viewport meta tag in HTML?

    • This tag renders the width of the page to the width of the device's screen size. If you have a mobile device that is 600px wide, then the browser window will also be 600px wide.

  8. What does X-UA-Compatible mean?

    • This < meta > tag specifies the document mode for Internet Explorer. IE=edge is the highest supported mode.

  9. What are HTML title tags?

    • The < title > tag is the title for the web page. This text is shown in the browser's title bar.

  10. What is the body tag?

    • The < body > tag in HTML is used to define the main content present inside an HTML page.
